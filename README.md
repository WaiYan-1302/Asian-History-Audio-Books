# Japan Reading Audiobook Site

This folder is ready for GitHub Pages.

## Files
- `index.html` — book-style reader with sentence highlighting
- `Japan_reading_audiobook.mp3` — audiobook audio
- `timings.json` — sentence-level timing data
- `Japan_part_exact_text.txt` — source text used for the page

## How to upload to GitHub Pages
1. Create a new GitHub repository, for example `japan-reading-audiobook`.
2. Upload all files from this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch `main` and folder `/root`.
6. Save. GitHub will give you a public Pages URL.

## Timing note
The page uses sentence-level timing generated by detecting pauses in the uploaded MP3 and mapping them to the reading text.
